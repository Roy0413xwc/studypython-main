### 1. 计算机只认识什么？

计算机只认识 0 和 1。

---

### 2. Python 解释器的作用

- 将 Python 代码翻译成计算机能识别的 0 和 1，并交给计算机执行。
- 在解释器环境内可以一行一行地执行我们输入的代码。
- 也可以使用解释器程序去执行 `.py` 代码文件。

---

### 3. “.py” 文件是什么？

`.py` 文件是 python 语言的代码文件，里面记录了 python 的代码。

---

### 4. Python 解释器程序在哪？

通常在 `<Python安装目录>/python.exe`。
